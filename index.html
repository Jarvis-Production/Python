Переменные и типы данных

Теория

В Python существует несколько основных типов данных:

  • int - целые числа (1, 2, -5)
  • float - числа с плавающей точкой (1.5, -0.2)
  • str - строки ("Привет", 'Python')
  • bool - логический тип (True, False)

Переменные в Python создаются при первом присваивании. Тип указывать не нужно!

Практика

# Примеры создания переменных
name = "Анна"          # строка
age = 25              # целое число
height = 1.75         # число с плавающей точкой
is_student = True     # логическое значение

# Проверка типа переменной
print(type(name))     # 
print(type(age))      # 

# Преобразование типов
str_number = "42"
number = int(str_number)  # преобразование строки в число
print(number + 8)     # 50

Основные операторы

Теория

Арифметические операторы:

  • + - сложение
  • - - вычитание
  • * - умножение
  • / - деление
  • // - целочисленное деление
  • % - остаток от деления
  • ** - возведение в степень

Операторы сравнения:

  • == - равно
  • != - не равно
  • > - больше
  • < - меньше
  • >= - больше или равно
  • <= - меньше или равно

Практика

# Арифметические операции
x = 10
y = 3

print(x + y)    # 13
print(x - y)    # 7
print(x * y)    # 30
print(x / y)    # 3.3333...
print(x // y)   # 3
print(x % y)    # 1
print(x ** y)   # 1000

# Операторы сравнения
a = 5
b = 7
print(a == b)   # False
print(a != b)   # True
print(a > b)    # False
print(a < b)    # True

Условные операторы if-else

Теория

Условные операторы позволяют выполнять код в зависимости от условий:

  • if - если условие истинно
  • elif - иначе если (можно использовать несколько раз)
  • else - иначе (если все предыдущие условия ложны)

В Python для обозначения блоков кода используются отступы (обычно 4 пробела)

Практика

# Простое условие
age = 18
if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")

# Множественные условия
score = 85
if score >= 90:
    print("Отлично!")
elif score >= 75:
    print("Хорошо!")
elif score >= 60:
    print("Удовлетворительно")
else:
    print("Нужно подтянуть знания")

Циклы while и for

Теория

Цикл while:

Выполняется, пока условие истинно

Цикл for:

Используется для перебора последовательностей (списков, строк, диапазонов)

Функция range(n) создает последовательность чисел от 0 до n-1

Практика

# Цикл while
count = 0
while count < 5:
    print(count)
    count += 1

# Цикл for с range
for i in range(5):
    print(i)

# Цикл for со строкой
for char in "Python":
    print(char)

# Цикл for со списком
fruits = ["яблоко", "банан", "апельсин"]
for fruit in fruits:
    print(fruit)

Списки и кортежи

Теория

Списки (list):

Изменяемые последовательности элементов. Создаются с помощью []

Основные методы списков:

  • append() - добавить элемент в конец
  • insert() - вставить элемент по индексу
  • remove() - удалить элемент по значению
  • pop() - удалить и вернуть элемент по индексу
  • sort() - сортировка списка

Кортежи (tuple):

Неизменяемые последовательности. Создаются с помощью ()

Практика

# Работа со списками
numbers = [1, 2, 3, 4, 5]
numbers.append(6)      # [1, 2, 3, 4, 5, 6]
numbers.insert(0, 0)   # [0, 1, 2, 3, 4, 5, 6]
numbers.remove(3)      # [0, 1, 2, 4, 5, 6]
last = numbers.pop()   # last = 6, numbers = [0, 1, 2, 4, 5]

# Срезы списков
print(numbers[1:4])    # [1, 2, 4]
print(numbers[::-1])   # [5, 4, 2, 1, 0]

# Кортежи
point = (10, 20)
x, y = point          # распаковка кортежа

Словари

Теория

Словари (dict) - коллекции пар ключ-значение

Основные методы словарей:

  • keys() - получить все ключи
  • values() - получить все значения
  • items() - получить пары ключ-значение
  • get() - безопасное получение значения

Практика

# Создание словаря
person = {
    "name": "Иван",
    "age": 25,
    "city": "Москва"
}

# Доступ к элементам
print(person["name"])          # Иван
print(person.get("age"))       # 25
print(person.get("email", "не указан"))  # не указан

# Изменение и добавление
person["age"] = 26
person["email"] = "ivan@example.com"

# Перебор словаря
for key in person.keys():
    print(key)

for key, value in person.items():
    print(f"{key}: {value}")

Функции

Теория

Функции определяются с помощью ключевого слова def

Особенности функций в Python:

  • Могут возвращать несколько значений
  • Поддерживают параметры по умолчанию
  • Поддерживают именованные аргументы
  • Могут принимать произвольное число аргументов

Практика

# Простая функция
def greet(name):
    return f"Привет, {name}!"

# Функция с параметром по умолчанию
def power(x, n=2):
    return x ** n

# Функция с множеством возвращаемых значений
def divide(a, b):
    return a // b, a % b

# Функция с произвольным числом аргументов
def sum_all(*args):
    return sum(args)

# Примеры использования
print(greet("Анна"))          # Привет, Анна!
print(power(2))               # 4
print(power(2, 3))           # 8
quotient, remainder = divide(10, 3)
print(sum_all(1, 2, 3, 4))   # 10

Работа с файлами

Теория

В Python есть несколько режимов открытия файлов:

  • 'r' - чтение (по умолчанию)
  • 'w' - запись (создает новый файл)
  • 'a' - добавление в конец
  • 'b' - бинарный режим

Всегда используйте конструкцию with для работы с файлами - она автоматически закроет файл

Практика

# Запись в файл
with open('example.txt', 'w', encoding='utf-8') as file:
    file.write('Привет, мир!\n')
    file.write('Это новая строка')

# Чтение всего файла
with open('example.txt', 'r', encoding='utf-8') as file:
    content = file.read()
    print(content)

# Чтение построчно
with open('example.txt', 'r', encoding='utf-8') as file:
    for line in file:
        print(line.strip())

# Добавление в файл
with open('example.txt', 'a', encoding='utf-8') as file:
    file.write('\nДобавляем новую строку')

Обработка исключений

Теория

Исключения помогают обрабатывать ошибки в программе:

  • try - блок кода, где может возникнуть ошибка
  • except - обработка конкретной ошибки
  • else - выполняется, если ошибок не было
  • finally - выполняется всегда

Практика

# Базовая обработка исключений
try:
    number = int(input("Введите число: "))
    result = 100 / number
    print(f"Результат: {result}")
except ValueError:
    print("Ошибка: введите корректное число")
except ZeroDivisionError:
    print("Ошибка: деление на ноль")
except Exception as e:
    print(f"Произошла ошибка: {e}")
else:
    print("Операция выполнена успешно")
finally:
    print("Завершение программы")

# Создание собственного исключения
class MyError(Exception):
    def __init__(self, message):
        self.message = message

# Использование собственного исключения
def check_age(age):
    if age < 0:
        raise MyError("Возраст не может быть отрицательным")

Объектно-ориентированное программирование

Теория

ООП основано на трех принципах:

  • Инкапсуляция - объединение данных и методов
  • Наследование - создание новых классов на основе существующих
  • Полиморфизм - разное поведение одного и того же метода

Практика

# Создание класса
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        pass

# Наследование
class Dog(Animal):
    def speak(self):
        return f"{self.name} говорит: Гав!"

class Cat(Animal):
    def speak(self):
        return f"{self.name} говорит: Мяу!"

# Использование классов
dog = Dog("Бобик")
cat = Cat("Мурка")

print(dog.speak())  # Бобик говорит: Гав!
print(cat.speak())  # Мурка говорит: Мяу!

# Приватные атрибуты и методы
class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # приватный атрибут
    
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
            
    def get_balance(self):
        return self.__balance

Декораторы и генераторы

Теория

Декораторы:

Функции, которые модифицируют поведение других функций

Генераторы:

Функции, которые генерируют последовательность значений

Практика

# Декоратор для измерения времени выполнения
import time

def measure_time(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"Время выполнения: {end - start} секунд")
        return result
    return wrapper

@measure_time
def slow_function():
    time.sleep(1)
    return "Готово!"

# Генератор чисел Фибоначчи
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# Использование генератора
for number in fibonacci(10):
    print(number)

Работа с JSON и API

Теория

JSON (JavaScript Object Notation) - популярный формат обмена данными

Основные методы:

  • json.dumps() - преобразование Python в JSON
  • json.loads() - преобразование JSON в Python

Практика

import json
import requests

# Работа с JSON
data = {
    "name": "Иван",
    "age": 25,
    "cities": ["Москва", "Санкт-Петербург"]
}

# Сохранение в JSON
json_string = json.dumps(data, ensure_ascii=False, indent=4)
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=4)

# Чтение из JSON
with open('data.json', 'r', encoding='utf-8') as f:
    loaded_data = json.load(f)

# Работа с API
response = requests.get('https://api.example.com/data')
if response.status_code == 200:
    api_data = response.json()
    print(api_data)
